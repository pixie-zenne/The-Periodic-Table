## The Periodic Table

**'The Periodic Table'** is a terminal-based program designed to provide users with interactive access to information about chemical elements. Users can easily select an element by its atomic number to view details like its name, symbol, atomic mass, group, period, and electronegativity value.

### Developer's Note

The periodic table is a terminal-based program. With the periodic table printing to the terminal, the table will either print neatly or not becuase every user have a different monitor, font, font size, and configuration of their tabs. If the table did not print neatly and that you want to see the table in its proper form, you can either make the tabs have a smaller number of spaces or make the terminal's font smaller. But take it as a suggestion, your device, your choice.

### Features

- **Element Selection:** Users can choose elements by their numbers in the table.
- **Detailed Element Information:** Displays essential details including atomic number, atomic mass, and electron configuration.
- **User Interface:** Simple terminal visuals mimic the layout of the periodic table.

### Usage

1. Execute the program.

2. The periodic table will display in the terminal.

3. Enter the atomic number of the desired element to retrieve its information.

4. To exit the program, type `"exit"`.
